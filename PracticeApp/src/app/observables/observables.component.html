so now the difference between observable and promise we can understand by creating custom observable and promise:
  • We have both custom promise and custom observable in ts file and put them on running after every second it will keep on running
  • you can see console window and observe 'hello' is written only once (which is promise) while 'hi' is written 8 times (which is observable)
  • one promise object will always return data one times doesnt matter how many times you call it.
  • one observable can return the data everytime you call it.
  • now to stop observable on keep on sending us data we need to use 'unsubscribe', so in ts we have written code that when we have got response till 8 sec now dont send us data.
