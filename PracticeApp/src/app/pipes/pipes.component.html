Pipes

  • In-build Pipes

    • Example:

    Anything Written inside this input field will be converted into Uppercase using uppercase inbuild pipe

    -----------------

    After Adding uppercase inbuild pipe: {{inputText|uppercase}}

  • Custom pipes

    • Example:

    Both number and string will be sorted using custom pipe sorting

    Actual Input:

    {{inputNum}}

    {{inputStr}}

    --------------------

    After Sorting:

    {{inputNum|sorting:'number'}}

    {{inputStr|sorting}}
